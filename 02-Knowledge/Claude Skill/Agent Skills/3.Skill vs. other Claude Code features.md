#### Key takeaways
https://anthropic.skilljar.com/introduction-to-agent-skills/434528

- **CLAUDE.md** loads into every conversation and is best for always-on project standards. 
	- <mark style="background:#d3f8b6">Skills load on demand and are best for task-specific expertise</mark>
- **Subagents** run in isolated execution contexts — use them for delegated work. 
	- <mark style="background:#d3f8b6"><font color="#000000">Skills add knowledge to your current conversation</font></mark>
- **Hooks** are event-driven (fire on file saves, tool calls). 
	- <mark style="background:#d3f8b6">Skills are request-driven (activate based on what you're asking)</mark>
- **MCP servers** provide external tools and integrations — a different category entirely from skills

Each feature handles its own specialty — **combine them** rather than forcing everything into one approach

Claude Code offers several customization options: Skills, CLAUDE.md, subagents, hooks, and MCP servers. They solve different problems, and knowing when to use each prevents you from building the wrong thing. Let's break them down.
